# Lost In The Depths

A classic, turn-based roguelike built in the spirit of traditional dungeon crawlers and based on the [Yet Another Roguelike Tutorial](https://www.rogueliketutorials.com/).

Descend into shifting caverns, battle unsettling creatures, collect magic items and potions, and survive as long as you can... if the depths allow it.

## About The Game

Lost in the Depths is a minimalist roguelike focused on clarity, strategy, and tight gameplay loops.

It has:
* Procedural dungeon generation
* Turn-based combat
* Permadeath
* Inventory and item systems
* ASCII-style rendering

## Features

* Infinite Replayability – Every run generates new dungeon layouts, enemies, and item placements.
* Tactical Combat – Positioning, movement, and quick reflexes all matter.
* Straightforward Controls – Jump in quickly, master the depth slowly.
* Extendable Architecture – Easy to add new monsters, item types, and systems.
* Clean Codebase – Designed for learning, experimentation, and future modding.

## Technologies Used

* Python 3.8
* tcod (libtcod) for rendering, input, and pathfinding
* NumPy for dungeon generation support
